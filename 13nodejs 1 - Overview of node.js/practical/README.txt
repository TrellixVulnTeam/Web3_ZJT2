In order to run the nodejs1.ipynb file in a lab computer:

1. Open a command prompt in the folder where the nodejs1.ipynb is located
2. Type 'jupyter-notebook nodejs1.ipynb'
3. Ignore the warning about the lack of kernel and just proceed without one


In order to run the nodejs1.ipynb file in your own computer:

1. Go to https://www.continuum.io/downloads
2. Download a version of Python Anaconda for your computer platform
3. Install Anaconda (Anaconda is just a Python interpreter with a bunch of useful libraries superimposed on top of it)
4. Open a command prompt in the folder where the nodejs1.ipynb is located
5. Type 'jupyter-notebook nodejs1.ipynb'
6. Ignore the warning about the lack of kernel and just proceed without one
