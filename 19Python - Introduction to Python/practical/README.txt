In order to run the Python practical notebooks, install the Anaconda Python distribution in your personal computer

https://www.continuum.io/downloads

(This distribution is installed by default in the OP computers image)

Once the Anaconda Python distribution is installed, you should be able to click in *.ipynb files and fire up the the corresponding Jupyter notebook.

The great advantage of-notebooks is that you can run illustrative code snippets within the notebook itself by simply clicking on the code snippet and pressing:
Ctrl+Enter
